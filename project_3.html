Overview of Portfolio Project

I re-created the frontend of portfolio website, inspired by the innovative 'Dopefolio' template. This open-source masterpiece, created by a talented developer, provided the foundation for my vision. The 'Dopefolio' template set the standard with its modern, clean design and user-friendly interface, making it an excellent choice for showcasing my work.

In this project, i have re-created the Portfolio Template with Html, CSS, Bootstrap ,Sass and JavaScript. I have used modern css like grid, flexbox, media queries etc.

Explore my portfolio and experience the seamless fusion of inspiration and originality.
