Overview of the Tesla Clone Project

In this project, I made a copy of Tesla's website by carefully remaking its look. I used modern technologies to create a similar and attractive user experience. This improved my web development skills and gave me insights into the latest frontend technologies.

The Tesla Clone project was an exhilarating challenge that allowed me to immerse myself in the world of web development and design. My goal was to replicate the look and feel of the original website, offering users a familiar and seamless interface.

In this project, i have re-create the Tesla landing page from scratch with Html, CSS and JavaScript. I have used modern css like grid, flexbox, media queries etc.
