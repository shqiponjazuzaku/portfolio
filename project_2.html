Overview of the Moana's Template

Moana is a web development project I've crafted, focusing on the frontend design for an online clothing store. While Moana is not an operational e-commerce site, it serves as a showcase of my web development and design capabilities. This project reflects my commitment to creating engaging online experiences for shoppers in the fashion industry.

Moana is a web development project that includes essential elements of an e-commerce platform, featuring product listings, detailed product pages, and a responsive design for seamless user experiences across various devices.

In this project, i have created the Moana Template with Html, CSS, Bootstrap ,Sass and JavaScript. I have used modern css like grid, flexbox, media queries etc.
